# Production-Style DevOps

This repository contains production-oriented DevOps implementations
built as part of hands-on infrastructure and automation challenges.

The goal of this repository is to demonstrate practical, real-world
DevOps skills using Docker, Kubernetes, Jenkins, and Terraform.

---

## Technologies Covered

- Docker (Image building, optimization, container security)
- Kubernetes (Deployments, Services, scaling)
- Jenkins (CI/CD pipelines)
- Terraform (Infrastructure as Code)

---

## Current Implementations

### Docker
- Python application containerization
- Custom port mapping
- Non-root container execution
- Optimized slim image usage
